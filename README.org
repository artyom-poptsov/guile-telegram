* Guile-Telegram
  A library for Telegram bot API.

* Dependencies
  - CURL
  - [[https://github.com/NalaGinrut/guile-json][Guile-JSON]]

* Usage
** 'config.scm' file format for example bot
A list of following strings:
#+BEGIN_EXAMPLE scheme
(<token> <bot-name> <proxy>)
#+END_EXAMPLE
